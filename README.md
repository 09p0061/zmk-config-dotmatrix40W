# dotmatrix40W - ZMK Configuration

ワイヤレス40%オーソリニアキーボード「**dotmatrix40Wireless** (略称: **dotmatrix40W**)」用のZMKファームウェアリポジトリです。

## dotmatrix40W について

- **46キーのレイアウト**: 4行×12列をベースに、最下段は1.5Uサイズのキーを配置しています。
- **ロータリーエンコーダ搭載**: 中央に配置されたエンコーダを回すことで、スクロールやLEDの明るさ調整が直感的に行えます。
- **ワイヤレス対応 (Bluetooth)**: ケーブルなしでPCやスマートフォンと接続できます。
- **単三電池1本で動作**: 安全な乾電池に対応しており、数カ月の使用が可能です。
- **フィラメントLED**: 暖かみのあるバックライトを搭載し、明るさの調整も可能です。

## キーマップのカスタマイズ方法

dotmatrix40Wは、主に2つの方法でキーマップをカスタマイズできます。
※JIS配列に対応するためにMod-Morph機能を使用している関係上、GUIエディタ（ZMK StudioやKeymap Editor）上では一部のキーが正しく表示されない（または意図しないキー名で表示される）場合があります。

### 方法A: ZMK Studio

ブラウザ上でキーマップを変更できるツールです。
ただし、現在のZMK Studioの仕様上、ロータリーエンコーダの割り当てを変更することはできません。

### 1. USBで接続する
キーボードをPCにUSBケーブルで接続します（現在のZMK StudioはUSB接続時のみ対応しています）。

### 2. ZMK Studioを開く
Google ChromeやEdgeなど、Web Serial APIに対応したブラウザで [ZMK Studio](https://zmk.studio/) にアクセスします。

### 3. キーボードと連携する
画面の「Connect」ボタンを押し、リストからキーボードを選択して接続します。

### 4. キーマップを編集する
画面上でキー配置を変更できます。変更した内容は**キーボード本体に保存**されます。

---

### 方法B: Keymap Editor

GitHubの機能を利用し、ブラウザ上でファームウェア自体を作り直す方法です。
ご自身のGitHubアカウントが必要になります。

### 1. リポジトリをコピー（Fork）する
画面右上の **Fork** ボタンを押して、このリポジトリをご自身のGitHubアカウントにコピー（フォーク）してください。その後、リポジトリの **Actions** タブを開き、GitHub Actionsの実行を許可（Enable Workflows）してください。

### 2. Keymap Editorにアクセス
ブラウザで [Keymap Editor](https://nickcoutsos.github.io/keymap-editor/) を開きます。

### 3. GitHubと連携する
Keymap Editor上でGitHubアカウントを連携（Authorize）し、リストから先ほどForkした `zmk-config-dotmatrix40W` を選択します。

### 4. キーマップを編集する
画面上でドラッグ＆ドロップして、お好みのキー配置に変更します。

### 5. 保存してファームウェアをビルドする
編集が終わったら、Keymap Editor右上にある **Save** ボタンを押します。
自動的にGitHub上でビルド作業（ファームウェアの作成）が始まります。

### 6. ファームウェアを書き込む
ご自身のGitHubリポジトリの **Actions** タブを開き、完了したビルドからファームウェア（`.uf2` 形式のファイル）をダウンロードします。
キーボードをUSB接続した状態で、**リセットスイッチを素早く2度押し**すると、「DOTMTRX40W」というデバイスとして認識されるので、uf2ファイルをドラッグ＆ドロップすれば書き込み完了です。

## 初期のレイヤー設定

デフォルトでは以下の3つのレイヤーが設定されています（自由に書き換え可能です）。

- **Layer 0 (Base)**: 通常のタイピング用。エンコーダは「上下スクロール」。
- **Layer 1 (Fn & Number)**: F1〜F12キー、テンキー、記号入力。エンコーダは「左右スクロール」。
- **Layer 2 (System)**: Bluetooth設定関連。エンコーダは「フィラメントLEDの明るさ変更」。
